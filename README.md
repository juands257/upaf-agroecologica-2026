# UPAF Agroecológica

Sitio académico estático para presentar el Trabajo Práctico Integrador de Tecnologías de la Producción Agropecuaria.

## Archivos

- `index.html`: presentación de la UPAF y visor para la secuencia didáctica.
- `desarrollo.html`: dashboard académico con 9 tarjetas de recursos.
- `video.html`: página exclusiva para el video educativo final y créditos.
- `styles.css`: estilos responsive, paleta agroecológica y animaciones.
- `script.js`: menú móvil, animaciones y apertura de áreas embebibles.

## Uso local

Abrir `index.html` en el navegador. No requiere instalación ni servidor.

## Incrustar recursos

Reemplazar cada placeholder por un iframe, por ejemplo:

```html
<iframe src="URL_DEL_RECURSO" title="Título del recurso" loading="lazy" allowfullscreen></iframe>
```
